# Deputy.com Competitor Analysis

## Overview
Deputy is a market leader in workforce management with 385,000+ workplace users. This analysis documents their key features for Uplift feature parity.

---

## Scheduling Features

### Filter Types
1. **Location Filter** - Multi-select for different work sites
2. **Date Range** - Day / Week / 2-Week / Month views
3. **Department Filter** - Filter by team/department
4. **Shift Type** - Morning, Afternoon, Evening, Night
5. **Employee Status** - Available, On Leave, On Shift
6. **Skills/Qualifications** - Filter by required certifications

### Multi-Location Scheduling
- Area-specific assignments
- Cross-location visibility
- Location-based labor cost tracking
- Regional manager views

### AI/Auto-Scheduling Features
- "Create AI optimized schedules with a single click"
- Labor demand forecasting integration
- Real-time wage-vs-sales comparison
- Budget optimization
- Constraint-based scheduling:
  - Respect availability
  - Maximum hours per week
  - Required certifications
  - Break compliance

---

## Skills & Qualifications Tracking

### Approach
- Skills tracked as attributes on employee profiles
- Certifications with expiry dates
- Required skills flagged on shift templates
- Auto-matching employees to shifts based on skills

### Types Tracked
- Certifications (first aid, food safety, machinery)
- Licenses (personal license, driver's license)
- Training completions
- Equipment proficiencies

## Break & Compliance

## Key Differentiators (Deputy vs. Uplift)

| Feature | Deputy | Uplift Advantage |
|---------|--------|-----------------|
| Scheduling | Strong | Comparable |
| Skills Tracking | Basic | **STRONGER** - Detailed competency levels, career progression |
| Career Development | None | **UNIQUE** - Internal opportunities, skill gap analysis |
| AI Scheduling | Strong | Comparable with reasoning |
| Employee Engagement | Basic | **STRONGER** - Recognition, momentum scores |
| Multi-language | Limited | **STRONGER** - 48 languages |

---

## Recommendations for Uplift

1. **Match Deputy's filtering** - Ensure all filter types are implemented
2. **Exceed on skills** - This is our differentiator
3. **Show AI reasoning** - Deputy doesn't explain why assignments are made
4. **Career paths** - Deputy doesn't offer this at all
5. **Recognition system** - Deputy lacks employee engagement features
